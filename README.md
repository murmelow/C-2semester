# C-2semester
Лабораторные работы на C++ (2 семестр).

lab_1 - Сумма элементов массива и минимальный элемент и его индекс.

lab_2 - Создание новго массива на основе признака отбора положительных чисел с динамическим выделением памяти.

lab_3 - Список без повторений + рейтинг + подсчет суммарной характеристики.

lab_4 - Линейный и бинарный поиск в динамическом массиве символьных строк.

lab_5 - Деление строки на слова в случае разных разделителей.
